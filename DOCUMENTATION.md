# Project File Documentation (Non-HTML)

This document provides documentation for repository files that did not already have clear file-level documentation. HTML files are intentionally excluded.

## /.gitignore
Defines local and generated paths Git should ignore, including Python caches, virtual environments, IDE settings, and `.env` files.

## /package.json
Defines frontend package metadata, Vite scripts (`dev`, `build`, `preview`), and React/Tailwind/PostCSS dependencies.

## /package-lock.json
Lockfile generated by npm to pin exact dependency versions for reproducible installs.

## /postcss.config.js
Configures PostCSS plugins used in the build pipeline:
- `tailwindcss`
- `autoprefixer`

## /tailwind.config.js
Configures Tailwind CSS content scanning paths and theme/plugin extension points.

## /vite.config.js
Defines Vite build/dev server configuration, enabling the React plugin and setting dev server defaults.

## /grant.jsx
Legacy/alternate React Grant Hunter UI component with:
- disease code and state selectors
- local FPL preview calculation
- API call to `POST /api/hunt`
- result card rendering in-page

## /src/main.jsx
Frontend bootstrap file that mounts the React app into `#root` and imports global CSS.

## /src/App.jsx
Primary React application file containing:
- form page for disease/income/state inputs
- FPL preview logic
- API search trigger (`/api/hunt`)
- sessionStorage persistence for results
- hash-based form/results page switching

## /src/index.css
Global stylesheet entry point that loads Tailwind base, components, and utilities layers.

## /dist/
Contains generated production build artifacts created by Vite (`npm run build`), including compiled JS/CSS bundles and processed `index.html`.
